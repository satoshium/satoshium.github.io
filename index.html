Zclassic is a fork of Zcash.

  • No Founders Tax

    All mining rewards in Zclassic go to miners, unlike Zcash founders that take 20% of mining rewards for the first 4 years.

  • No Slow Start

    We changed the slow start parameter to be 10 blocks instead of 20,000 blocks in Zcash.

  • Trusted Parameters

    We are using the same trusted parameters Zcash uses.
